?ver Remover! Remove wp version param from any enqueued styles & scripts files.

== Description ==

The "WP Version Remover" plugin useful to remove wp version param from any enqueued scripts OR styles files. Sometime due to version parama( i.e ?ver) css/js update not reflect in front-end, In that case using "WP Version Remover" plugin we can remove version param easily.

Usign "WP Version Remover" plugin we can remove generator tag as well.

== Installation ==

Installing the plugin:

1.  Unzip the plugin's directory into `wp-content/plugins`.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
3.  The settings page will be available under settings menu (Settings >> WP Version Remover)
